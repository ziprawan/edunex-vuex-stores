# Edunex VueX Store Definitions

Definitions:

- Fetch `<method>` `<pathname>`: Make a request with method `<method>` to https://api-edunex.cognisia.id with the given `<pathname>` and queries (and body if said so), and then parse the response as JSON. If the request or JSON parsing fails, it throw error.

## States

Contains all VueX stores found in the app.js file. Some may be missing or missing in action due to the randomness of the code.
